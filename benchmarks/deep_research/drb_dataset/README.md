## English

### 📖 Dataset Overview

This is the official dataset accompanying the **[DeepResearch Bench](https://github.com/Ayanami0730/deep_research_bench)** paper. It contains research reports generated by 4 leading deep research AI systems along with detailed human expert annotations evaluating these reports.

DeepResearch Bench is the first comprehensive benchmark for systematically evaluating Deep Research Agents (DRAs) on their ability to handle complex, PhD-level research tasks.

**Paper**: [arxiv.org/pdf/2506.11763](https://arxiv.org/pdf/2506.11763)  
**Leaderboard**: [DeepResearch Bench Leaderboard](https://huggingface.co/spaces/muset-ai/DeepResearch-Bench-Leaderboard)  
**Main Repository**: [github.com/Ayanami0730/deep_research_bench](https://github.com/Ayanami0730/deep_research_bench)

### 📊 Dataset Contents

#### 1. Generated Research Reports (`generated_reports/`)

Contains complete research articles generated by 4 leading AI systems for 100 research tasks:

- **`gemini-2.5-pro-deepresearch.jsonl`** (100 records) - Google Gemini 2.5 Pro Deep Research
- **`grok-deeper-search.jsonl`** (100 records) - xAI Grok Deeper Search
- **`openai-deepresearch.jsonl`** (100 records) - OpenAI Deep Research
- **`perplexity-Research.jsonl`** (100 records) - Perplexity Research Mode

Each record format:
```json
{
  "id": 1,
  "prompt": "Original research task query...",
  "article": "Complete research article generated by the AI system, with citations..."
}
```

#### 2. Human RACE Annotations (`human_RACE_annotation.jsonl`)

Contains 150 expert human evaluation records using the RACE (Reference-based Adaptive Criteria-driven Evaluation) framework to assess reports from all 4 models.

Each record format:
```json
{
  "id": 1,
  "annotation_id": "unique_annotation_id",
  "dimension_scores": {
    "openai-deepresearch": {
      "Comprehensiveness": 80,
      "depth": 85,
      "instruction following": 80,
      "readability": 82
    },
    "perplexity-Research": {...},
    "grok-deeper-search": {...},
    "gemini-2.5-pro-deepresearch": {...}
  },
  "overall_scores": {
    "openai-deepresearch": 84,
    "perplexity-Research": 68,
    "grok-deeper-search": 42,
    "gemini-2.5-pro-deepresearch": 80
  }
}
```

### 📐 Evaluation Dimensions

The RACE evaluation framework assesses reports across four core dimensions:

1. **Comprehensiveness**: Coverage breadth and depth of the research topic
2. **Depth**: Quality of analysis and insight generation
3. **Instruction Following**: Adherence to specific task requirements
4. **Readability**: Clarity, organization, and presentation quality

All scores range from 0 to 100.

### 🎯 Research Task Characteristics

- **Task Count**: 100 high-quality research tasks
- **Difficulty Level**: PhD-level complex research questions
- **Domain Coverage**: 22 distinct fields including:
  - 🔬 Science & Technology (Physics, Chemistry, Biology, Environmental Science, Engineering)
  - 💼 Finance & Business (Investments, Personal Finance, Marketing, HR)
  - 💻 Software & Internet
  - 🌍 Others (Art & Design, Entertainment, History, Industrial, Transportation, Travel, etc.)
- **Language Distribution**: 50 Chinese tasks + 50 English tasks

### 💡 Use Cases

This dataset is valuable for:

- **Deep Research Agent Development**: Benchmarking and improving deep research AI systems
- **Model Comparison Studies**: Comparing different AI systems on complex research tasks
- **Evaluation Methodology Research**: Studying and improving evaluation methods for AI-generated long-form content
- **Education & Training**: Understanding capabilities and limitations of state-of-the-art AI research systems

## 中文说明

### 📖 数据集简介

本数据集是 **[DeepResearch Bench](https://github.com/Ayanami0730/deep_research_bench)** 论文的官方配套数据集，包含了4个顶尖深度研究AI系统生成的研究报告以及专业人类评估员对这些报告的详细评分。

DeepResearch Bench 是首个针对深度研究智能体（Deep Research Agents, DRAs）的综合性基准测试，旨在系统评估AI系统在执行复杂、PhD级别研究任务时的能力。

**论文链接**: [arxiv.org/pdf/2506.11763](https://arxiv.org/pdf/2506.11763)  
**排行榜**: [DeepResearch Bench Leaderboard](https://huggingface.co/spaces/deep-research/DeepResearch-Bench-Leaderboard)  
**主仓库**: [github.com/Ayanami0730/deep_research_bench](https://github.com/Ayanami0730/deep_research_bench)

### 📊 数据集内容

#### 1. 生成的研究报告 (`generated_reports/`)

包含4个领先AI系统针对100个研究任务生成的完整研究文章：

- **`gemini-2.5-pro-deepresearch.jsonl`** (100条记录) - Google Gemini 2.5 Pro 深度研究模式
- **`grok-deeper-search.jsonl`** (100条记录) - xAI Grok 深度搜索模式
- **`openai-deepresearch.jsonl`** (100条记录) - OpenAI 深度研究系统
- **`perplexity-Research.jsonl`** (100条记录) - Perplexity Research 模式

每条记录格式：
```json
{
  "id": 1,
  "prompt": "研究任务的原始问题...",
  "article": "AI系统生成的完整研究报告，包含引用来源..."
}
```

#### 2. 人类评估标注 (`human_RACE_annotation.jsonl`)

包含150条专业人类评估员对上述4个模型生成文章的RACE（Reference-based Adaptive Criteria-driven Evaluation）评分记录。

每条记录格式：
```json
{
  "id": 1,
  "annotation_id": "唯一标注ID",
  "dimension_scores": {
    "openai-deepresearch": {
      "Comprehensiveness": 80,
      "depth": 85,
      "instruction following": 80,
      "readability": 82
    },
    "perplexity-Research": {...},
    "grok-deeper-search": {...},
    "gemini-2.5-pro-deepresearch": {...}
  },
  "overall_scores": {
    "openai-deepresearch": 84,
    "perplexity-Research": 68,
    "grok-deeper-search": 42,
    "gemini-2.5-pro-deepresearch": 80
  }
}
```

### 📐 评估维度

RACE评估框架使用以下四个核心维度：

1. **Comprehensiveness (全面性)**: 研究主题覆盖的广度和深度
2. **Depth (深度/洞察力)**: 分析质量和洞察生成能力
3. **Instruction Following (指令遵循)**: 对特定任务要求的遵循程度
4. **Readability (可读性)**: 报告的清晰度、组织性和呈现质量

所有评分采用0-100分制。

### 🎯 研究任务特点

- **任务数量**: 100个高质量研究任务
- **难度级别**: PhD级别的复杂研究问题
- **领域覆盖**: 22个不同领域，包括：
  - 🔬 科学与技术（物理、化学、生物、环境科学、工程）
  - 💼 金融与商业（投资、个人理财、营销、人力资源）
  - 💻 软件与互联网
  - 🌍 其他领域（艺术设计、娱乐、历史、工业、交通、旅游等）
- **语言分布**: 50个中文任务 + 50个英文任务

### 💡 使用场景

本数据集适用于：

- **深度研究智能体开发**: 为开发和改进深度研究AI系统提供基准
- **模型对比研究**: 比较不同AI系统在复杂研究任务上的表现
- **评估方法研究**: 研究和改进AI生成长文本内容的评估方法
- **教学与培训**: 了解顶尖AI系统的研究能力和局限性
