# Katala Project Concept & Vision (v1.1)

## 1. 核心コンセプト (Core Concepts)
- **AI-Mediated Catalyst**: AIが「触媒 (Catalyst)」となり、人々の「語り」を誘発し、最適な繋がりを創出する。
- **Agent-to-Agent Surface**: ユーザー自身が繋がる前に、それぞれのAIエージェント（例：ニコラスの「しろくま」、しえらの「カニ」）が表面上で繋がり、相性や情報を事前確認する。
- **Profiling through Dialogue**: MBTI診断の進化系。AIとの対話を通じて、ユーザー自身も気づいていない価値観をプロファイリングし、動的にアップデートする。
- **Autonomous Synergy Scoring**: X (Twitter) のアルゴリズムを応用し、エンゲージメントだけでなく「人生の目的」や「ビジネス上の共鳴」をスコアリング。

## 2. 接続モード (Connection Modes)
- **Nexus Mode**: 未知のユーザーとシナジーベースで繋がる標準モード。
- **Close Friends Mode**: 信頼できる友人と、より深く・優先的にエージェントを同期させるモード。

## 3. インフラと技術スタック
- **Auth Layer**: Vercel (Auth.js / NextAuth) を基盤とした、摩擦のないセキュアなログイン体験。
- **Frontend**: Next.js (App Router) + Tailwind CSS + Framer Motion (Apple HIG準拠).
- **Backend**: Rust (Performance/Safety) - `x-algorithm` 統合.
- **AI/ML**: Claude 3.5/Opus, JAX/Haiku (Phoenix Model).
