# inf-Physics

物理学の式・関係式・基礎量・理論別コア方程式を保管する inf-Memory カテゴリ。

## 目的

- physics formula inventory を収集・保管する
- 相対論・量子論・場の理論・宇宙論などの式一覧を整理する
- Katala GUT の対応表・基本量辞書・統一式候補の参照元にする

## 構造

- `equations/` : 物理学の式一覧・辞書・対応表
- `raw/` : 生データ / 収集直後の資料
- `normalized/` : 正規化済みの式データ
- `indexed/` : 理論別・分野別・記号別索引
- `projections/` : theory-linked projection / summary view
- `curated/` : 人手検証済みセット
- `manifests/` : inventory / state / import log

## 現状

現時点ではカテゴリ構造のみ作成済み。
今後、相対論側基本量表・量子論側基本量表・物理学コア方程式一覧などをここへ保存する想定。
