# inf-Mathematical-Papers

数学論文コーパスを保管する inf-Memory カテゴリ。

## 目的

- peer-reviewed mathematical papers を収集・保管する
- raw / normalized / indexed / projection / curated の段階を分離する
- 後続の inf-Brain / Katala GUT / 幾何学カタログ化で、数学論文の由来と加工段階を追跡できるようにする

## 構造

- `raw/` : DOI harvest や論文メタデータの生データ
- `normalized/` : 正規化済みメタデータ
- `indexed/` : 分野・キーワード・著者・年代などの索引
- `projections/` : latest / topic / theory-linked projection
- `curated/` : 人手検証・精査済みセット
- `manifests/` : inventory / state / import log

## 現状

現時点ではカテゴリ構造のみ作成済み。
今後の数学論文収集ジョブは、このカテゴリを正式保存先として使う想定。
