> Generated: 2026-02-28
> Source: OpenAlex API + Semantic Scholar + Perplexity web search
> Scope: Cross-modal translation, music cognition, qualia measurement, aesthetic experience

## Overview

This survey collects key literature relevant to the **Human Translation Loss Framework (HTLF)**, organized by the three core ratios:

- **R_struct**: Structural preservation in cross-modal translation (音楽→言語→視覚 etc.)
- **R_context**: Contextual/cultural information preservation across domains
- **R_qualia**: Subjective experiential quality preservation (the "hard problem" of translation)

**Summary**: Shannon (1948) provides the mathematical foundation for R_struct: information is quantifiable, and channel capacity determines maximum lossless transmission. However, Shannon explicitly excluded semantics—HTLF extends his framework to semantic and experiential information. Goodman (1968) distinguishes "dense" symbol systems (like music, painting) from "articulate" ones (like language, notation), arguing that translation between dense and articulate systems necessarily loses information. This is the theoretical basis for R_struct < 1 in music→language translation. Steiner (1975) argues all translation involves "hermeneutic loss"—context-dependent meaning that resists transfer.

## Key Gaps Identified

1. **No existing framework** explicitly measures information loss across modalities with the structural/context/qualia decomposition proposed by HTLF
2. **Qualia measurement** remains indirect—no behavioral protocol directly quantifies subjective experience loss in translation
3. **Cross-modal translation studies** focus on correspondence (what maps to what) rather than **loss** (what doesn't survive translation)
4. **Computational translation metrics** (BLEU, etc.) ignore semantic and experiential dimensions entirely
5. **Embodied dimensions** of music cognition are underrepresented in translation loss studies

## Recommended Next Steps (Phase 1)

1. Develop pilot experiment measuring R_struct: music→verbal description→music reconstruction, measuring structural fidelity
2. Design R_qualia proxy using physiological measures (SCR, HR) comparing original vs. reconstructed stimuli
3. Formalize R_context using Nattiez's tripartite model (poietic/neutral/esthesic) with cultural background as controlled variable
4. Review IIT's Φ as candidate metric for theoretical R_qualia upper bound
