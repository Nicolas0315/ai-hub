# Katala Synergy Rewards & Growth Policy

## 1. Concept: "The Altruistic Synergy Score"
- **周囲への貢献の可視化**: 自分のエージェントが、他者やネットワーク全体のシナジー最大化にどれだけ寄与したかを「Synergy Contribution Score (SCS)」として数値化する。
- **Agent Nurturing**: エージェントの知能や交渉能力だけでなく、「どれだけ他者を助け、良いマッチングを生み出したか」という「徳」のパラメータを導入する。

## 2. Reward Mechanism
- **Contribution Rewards**: SCSが高いユーザーに対し、計算リソースの優先割り当て、ネットワーク内での限定的な権限、あるいはエコシステム内のインセンティブを付与する。
- **Social Credit for Agents**: 信頼性が高く、周囲に利益をもたらすエージェントほど、他者のエージェントから「優先的な交渉相手」として選ばれやすくなる。

## 3. Agent Development as a Skill
- **The Nurturer’s Advantage**: エージェントに正確な SOUL.md を刻み、適切に情報を Staging し、世界に貢献させる「飼い主（ユーザー）」のスキルが、ネットワーク内での優位性に直結する仕組み。
- ユーザーは「エージェントを育てる」ことで、間接的に世界に良い影響を与え、その見返りを得る。

## 4. Implementation Strategy
- `SynergyEngine` に、マッチング成功時の「貢献度分配ロジック」を実装。
- 「板（The Board）」に、現在の高貢献ユーザーやエージェントのランキングを「信頼の指標」として表示する。
